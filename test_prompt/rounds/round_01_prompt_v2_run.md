# Round 01 - prompt_v2_run

## 1. 本轮目标

- 用 `prompt_v2` 在固定样本上完成第一轮小批量复跑
- 验证抽取脚本、DeepSeek 接口、输出结构是否稳定

## 2. 复跑配置

- Prompt 版本：`v2`
- 模型：`deepseek-reasoner`
- Base URL：`https://api.openai-proxy.org/v1`
- 书目：Book 04、Book 09
- Chunk：`5,6,7,8`
- 运行结果：8/8 成功，0 失败

## 3. 初步观察

### 实体

- Book 04 chunk 5 的实体数量收敛到了 4 个，明显比旧版本更克制
- Book 04 chunk 6 仍然出现同义词/近义项重复：`百分位数 / 分位数`、`箱线图 / 盒须图`
- Book 09 chunk 8 仍把 `客观性`、`计算效率` 一类评测维度抽成实体，说明 v2 约束还不够硬
- Book 04 chunk 7 抽取为空，需要人工判断是真空段还是漏抽

### 关系

- 关系总量比容易失控的版本更收敛
- 仍有不该保留的关系，例如把同义词或近义表达连成 `衍生`
- `包含` 方向仍有风险，不能认为已解决

### 定义

- 定义总体比旧版本更像定义
- 但仍有区分不够清的问题，例如 `方差` 和 `标准差` 的定义过近

## 4. 本轮结论

- `prompt_v2` 可以保留为有效中间版本
- 但质量还未达标，不适合直接扩展到任务10
- 下一步需要引入“第二个助手”做批量复审，快速给出驳回意见和 prompt 修改方向
