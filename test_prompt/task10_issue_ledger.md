# Task10 Issue Ledger

这个文件用于累计记录任务10批量抽取中出现的问题，避免每轮重复讨论却没有沉淀。

## 使用规则

- 每发现一个“值得驱动 prompt 修改”的问题，就记一条
- 只记高频问题、结构性问题、会影响批量质量的问题
- 不要把一次性的 API 错误或偶然坏样本混进来
