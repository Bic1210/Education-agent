# 人工评估 Checklist

每次复跑后，至少检查下面这些点。

## 实体

- [ ] 实体名称在原文中明确出现
- [ ] 不是人名、地名、数字、表格列名、案例细节
- [ ] 不是评测指标、评价维度、优缺点、属性词
- [ ] 不是明显过于具体或只在本段临时出现的词
- [ ] 类型标注合理
- [ ] 定义不是“某种方法/某个概念”这类空话
- [ ] 定义没有和其他实体混淆

## 关系

- [ ] 关系能被原文直接支持
- [ ] 不是常识脑补出来的关系
- [ ] 不是为了连线而连线的弱关系
- [ ] `包含` 方向正确
- [ ] `对比` 不是用于同义词
- [ ] `衍生` 不是误代替“是一种”

## 整体

- [ ] 实体数量没有明显失控
- [ ] 关系数量没有明显失控
- [ ] 结果没有过度概括
- [ ] 结果没有过度解读
- [ ] 和上一轮相比确实更稳
